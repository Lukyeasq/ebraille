Directions

When submitting a tactile graphic for use in development of the E-braille specification, please make a copy of this template and fill in all items marked with "xxx" and rename the file to be the same as the graphic, i.e., a file with the name of "duck.jpg" would also have a file called "duck.html" in the same folder. The "img" and the "a" markup is commented out, so remove the commented out markup. There is an example of commented out markup below this paragraph, but you will only see it in a text editor and a browser will hide it.

contributor: xxx

Organization making contribution: xxx

Tactile file name: xxx

if SVG, JPG, or PNG, please add the image using normal "img" HTML coding.

If there is an extended description (long description), please put it in the details element. You may edit the summary to meet the context of the image. The aria-details should include the id of the details element.

If providing a PDF, please use the ancor "a" HTML element.
